BoardingJourneyCard
Class that represents a generic boarding pass for any type of transport. This class contains attributes and example methods that could be considered common to any boarding card of any type of transport. Dice that the BaordingCard class could contain many attributes, many of which could be other complex objects, such as a vehicle, a airport, a train or bus station, the person who owns the boarding card, etc; you could choose to apply the Builder pattern for build objects of type BoardingJourneyCard, but given that this type of objects they need to be instantiated with parameters specified at the time of execution, I have opted for a provisional solution, to make the methods of this class that modify the values ​​of their attributes, return this, that is, the object that calls the method. In this way, we can chain calls to different Set methods, and avoid having several overloaded builders. Developed By : mostafa.monib@gmail.com mostafa monib
